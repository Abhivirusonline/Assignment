Q2: Explain difference between Bind and Call (example).

Call :

In Javascript all functions are object methods. The call() method is a predefined JavaScript method. It can be used to call a method with an owner object as an argument. With call() is used when the invocation of function is required immediately after modification of context.

Example :

var person = {
name: "Jitender Singh",
hello: function(thing) {
console.log(this.name + " says hello " + thing);
}
}
person.hello("world");
person.hello.call({ name: "Abhishek Yadav" }, "world");

Bind :

In Javascript all functions are object methods. The bind() method is a predefined JavaScript method. It can be used to call a method with an owner object as an argument. With bind() is used when the invocation of function is not required immediately after modification of context.

Example :

var person = {
name: "Jitender Singh",
hello: function(argument) {
console.log(this.name + " says hello " + argument);
}
}
person.hello("world");
var helloFunc = person.hello.bind({ name: "Abhishek Yadav" });
helloFunc("world");
